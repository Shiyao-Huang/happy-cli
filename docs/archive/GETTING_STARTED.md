# 🚀 Aha - 让 Claude Code 无处不在

## 为什么选择 Aha？

想象一下：
- 🛋️ 躺在沙发上用手机写代码
- 🚇 地铁上用平板电脑 review PR
- ☕ 咖啡厅里用笔记本远程控制家里的 Mac 写代码
- 🤝 一个人拥有一支 AI 团队（Master、Builder、QA、Architect...）

**这就是 Aha 带给你的体验！**

## ⚡ 30 秒上手

### 第一步：安装
```bash
npm install -g cc-aha-cli
```

### 第二步：启动守护进程
```bash
aha daemon start
```

### 第三步：打开 Web 应用
访问 **https://aha-agi.com/webapp**

**就这样！** 🎉 现在你可以在任何设备上控制 Claude Code 了！

## 🎯 核心特性

### 1. 📱 真正的移动编程体验
```
手机 ↔️ Claude Code
平板 ↔️ Claude Code
浏览器 ↔️ Claude Code
```
不再局限于开发机前，代码跟着你走！

### 2. 🤖 内置 22 个专业 AI 角色
无需配置，开箱即用的多智能体团队：

| 角色 | 职责 | 使用场景 |
|-----|------|---------|
| 🎯 **Master** | 项目协调、任务分配 | 管理整个开发流程 |
| 🔨 **Builder** | 核心开发、功能实现 | 快速构建功能 |
| 🏗️ **Architect** | 架构设计、技术决策 | 系统设计评审 |
| 🧪 **QA** | 质量保证、测试 | 自动化测试、bug 检测 |
| 🎨 **Framer** | UI/UX 实现 | 前端界面开发 |
| 🔍 **Scout** | 代码探索、问题定位 | 快速了解代码库 |
| 📝 **Scribe** | 文档编写 | API 文档、使用指南 |

...还有 15+ 个专业角色！

### 3. 🔒 端到端加密
所有通信使用 **TweetNaCl** 加密，数据安全无忧：
```
你的设备 --[加密]--> Aha 服务器 --[加密]--> Claude Code
```

### 4. 🌍 零配置团队协作
```bash
# 启动守护进程后，在 Web 端创建团队
# 自动生成 Master、Builder、QA 等角色
# 每个角色都是独立的 Claude 实例
# 自动协作完成复杂任务
```

## 🌟 为什么 Aha 与众不同？

### vs. 传统开发
| 传统方式 | Aha 方式 |
|---------|---------|
| 必须坐在电脑前 | 任何设备都能写代码 |

| 延迟高 | 实时同步 |

